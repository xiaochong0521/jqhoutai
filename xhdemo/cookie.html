XH框架自带cookie插件,不用另下载cookie插件,调用简单。
设置cookie调用$.xh_setCookie(name, value, time, path);
name: cookie名;
value: cookie值;
time: 设置过期时间,最低设置1小时，过期设置-1即可;
path: 路径;

获取cookie调用$.xh_getCookie(name);
name: cookie名;
code
  1. <script type="text/javascript">
  2.     // 设置cookie
  3.     $.xh_setCookie('xh', '测试123', 1, '/xhdemo');
  4.     // 获取cookie
  5.     $.xh_getCookie('xh');
  6. </script>
