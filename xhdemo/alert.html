警告框主题
数据提交成功！
警告！数据丢失
数据提交失败！
想让它隐藏起来，可以调用$('xxx').xh_alert('hide'),反之，显示为：$('xxx').xh_alert('show')
提示框
使用$('xxx').xh_prompt(prompt, text, time);
prompt:类型，必填
text：提示信息，必填
time：从显示到结束的时间，选填，不填则默认1.5秒(1000=1秒)
code
  1. <script type="text/javascript">
  2.    $('#prompt-success').click(function() {
  3.       $(this).xh_prompt('success', '数据提交成功', 2000);
  4.    });
  5.    $('#prompt-warm').click(function() {
  6.       $(this).xh_prompt('warm', '警告！数据丢失', 2000);
  7.    });
  8.    $('#prompt-danger').click(function() {
  9.       $(this).xh_prompt('danger', '数据提交失败！', 2000);
  10.    });
  11. </script>
弹窗框(加载模式)
调用xh_alert_load(start,text)即可加载;
start: 状态，show | hide 必填；text: 提示信息，选填
code
  1. <script type="text/javascript">
  2. $('#alert-load').click(function() {
  3.     xh_alert_load('show', '加载中');
  4.     setTimeout(function() { // 3秒后关闭
  5.         xh_alert_load('hide');
  6.     }, 3000);
  7. });
  8. </script>
按钮加载(一般用于提交数据)
使用$(xxx).xh_bload(text, isload);
text: 加载提示 必填;
isload: true | false 选填，默认为true,也就是加载状态
code
  1. <script type="text/javascript">
  2. $('#button-load').click(function() {
  3.     $(this).xh_bload('保存中', true);
  4.     var obj = this;
  5.     setTimeout(function() {    // 3秒后结束加载
  6.         $(obj).xh_bload('保存', false);
  7.     }, 3000);
  8. });
  9. </script>
confirm(一般用于防止用户点错)
简单好用，代码如下：
code
  1. <script type="text/javascript">
  2. $('#alert-load').click(function() {
  3.   var c = new xh_confirm({
  4.     title: '测试标题', // 非必填，不填则显示“操作提示”
  5.     content: '确定要解绑？', // 提示信息
  6.     becomes: true, // true 可以进行放大缩小
  7.     confirm_success: function() {
  8.       // 点击确定后，执行的地方
  9.       $(this).xh_prompt('success', '已解绑', 2000);
  10.     }
  11.   });
  12.   });
  13. </script>
模拟框
使用$(xxx).xh_modal(isshow);
isshow：show | hide 必填
show 显示，hide 隐藏；
