按钮风格个人自己制作，如果需要什么样得可以提出；引用方式：向任意HTML元素设定class="xh-btn"，建立一个基础按钮。内置的按钮class可以进行任意组合，从而形成更多种按钮风格。

按钮组
将按钮放入一个class="xh-btn-group"元素中，即可形成按钮组，按钮本身仍然可以随意搭配 
code
  1. <div class="tab-content xh-btn-group">
  2. <button class="xh-btn xh-btn-info><i class="fa fa-plus"></i>添加</button>
  3. <button class="xh-btn xh-btn-info><i class="fa fa-edit"></i>编辑</button>
  4. <button class="xh-btn xh-btn-info><i class="fa fa-trash-o"></i>删除</button>
  5. </div>
