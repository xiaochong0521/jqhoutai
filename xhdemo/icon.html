XH 的图标从网上采集的，本框架大部分引用Font Awesome图标字体库，Font Awesome图标字体库内容丰富，图标精美，操作多样简单，具体可以去Font Awesome官网上看。
最简单的引用方式： 
code
  1. <link rel="stylesheet" href="path/to/font-awesome/css/font-awesome.min.css">
固定宽度：使用 fa-fw 可以将图标设置为一个固定宽度。主要用于不同宽度图标无法对齐的情况。 尤其在列表或导航时起到重要作用。 
code
  1. <i class="fa fa-home fa-fw"></i>
动画：使用 fa-spin 类来使任意图标旋转，现在您还可以使用 fa-pulse 来使其进行8方位旋转。尤其适合 fa-spinner、fa-refresh 和 fa-cog 。 
code
  1. <i class="fa fa-home fa-spin"></i>
组合使用：如果想要将多个图标组合起来，使用 fa-stack 类作为父容器， fa-stack-1x 作为正常比例的图标， fa-stack-2x 作为大一些的图标。还可以使用 fa-inverse 类来切换图标颜色。您可以在父容器中 通过添加 大图标 类来控制整体大小。 
code
  1. <span class="fa-stack fa-lg">
  2.     <i class="fa fa-square-o fa-stack-2x"></i>
  3.     <i class="fa fa-twitter fa-stack-1x"></i>
  4.     <i class="fa fa-flag fa-stack-1x fa-inverse"></i>
  5.     <i class="fa fa-ban fa-stack-2x text-danger"></i>
  6. </span>
旋转与翻转：使用 fa-rotate-* 和 fa-flip-* 类可以对图标进行任意旋转和翻转。 
code
  1. <i class="fa fa-shield fa-rotate-90"></i>
  2. <i class="fa fa-shield fa-rotate-180"></i>
